# FODCA
fx67ll One Data Center App 
这是fx67ll个人日常使用的uni-app应用，[访问地址](https://life.fx67ll.com)  
暂时仅支持游客查看示例不支持账号注册，如果有使用兴趣可以在我的个人博客留言区联系我~ 

*Tip: 运行请先安装 `moment` & `underscore`*
